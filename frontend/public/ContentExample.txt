<h2><strong>Giới thiệu TipTap Editor</strong></h2>

<p>
TipTap là một <strong>Rich Text Editor</strong> mạnh mẽ dành cho React, được xây dựng trên <em>ProseMirror</em>.
Nó cho phép bạn tạo trình soạn thảo hiện đại như Notion hoặc Medium.
</p>

<img src="https://i.ytimg.com/vi/VcnROkRhJ34/maxresdefault.jpg" alt="TipTap Logo" title="TipTap Logo"/>

<p>Để bắt đầu, bạn có thể xem tài liệu chính thức tại:
  <a href="https://tiptap.dev" target="_blank" style="color: #098be4;">https://tiptap.dev</a>
</p>

<h3>Ví dụ khởi tạo TipTap</h3>

<pre><code class="language-js">
import { EditorProvider, EditorContent } from '@tiptap/react'
import StarterKit from '@tiptap/starter-kit'

const editor = new Editor({
  extensions: [StarterKit],
  content: '<p>Hello World!</p>',
})
</code></pre>

<p>Với vài dòng code trên, bạn đã có thể khởi tạo một editor cơ bản để nhập văn bản, in đậm, in nghiêng và tạo danh sách.</p>
